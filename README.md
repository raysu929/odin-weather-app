# Odin Weather App

A weather application built with JavaScript, HTML, and CSS, using Webpack for bundling.  

## Live Demo

Check out the live project here: [Odin Weather App](https://odin-weather-app-projec.netlify.app/)

## Features

- Displays current weather for any city
- Shows temperature, humidity, and weather conditions
- Responsive design for desktop and mobile

## Technologies Used

- JavaScript (ES6)
- HTML5 & CSS3
- Webpack
- Netlify for deployment
